Angular

Es un framework de node que tiene muchos archivos para crear un proyecto HTML+CSS+Js
Su filosofia se basa en trabajar con carpetas y que todo se separe en modulos (codigo) y componentes(codigo+vista)

Instalar: CMD

1 -> Instalar node.js DESCARGAR

2 -> Actualizar el manejador de paquetes de node: npm install -g npm@latest

3 -> Limpiar la cache de npm: npm cache clean --force

4 -> Desabilitar toda la seguridad: npm set audit false

Los siguientes pasos en caso de haberlo instalado antes:

5 -> Limpiar versión previa el CLI de angular: npm uninstall -g angular-cli

6 -> Desinstalar la antigua versión de angular: npm uninstall -g angular@cli

7 -> Limpiar de nuevo la cache: npm cache clean force

8 -> Instalar Angular: npm -g install @angular/cli@latest


Crear proyecto:

Crear la carpeta del proyecto, luego ir al CMD: ng new LoQueSeaMierda

Nota: No agregar el Routing de angular.

Nota: Recomiendo dejar el CSS.

Nota: El proceso es lento. debe descargar arvchivos y generar muchos ficheros.


Correr el Proyecto:

Ir al root y en CMD: ng serve

http://localhost:PORT/

Nota: no se puede cerrar el CMD


Estructura de los archivos:

      * Todo se basa en componentes y modulos:
      (Componente es el conjunto de vista, css, .ts)
      (Modulo es solo para funcionalidad .ts)
      LoQueSeaMierda.app.component.html = es la vista.
          >> Se maqueta totalmente igual y los textos de variables se ponen entre {{   }}
      LoQueSeaMierda.app.component.css = es el estilo.
      LoQueSeaMierda.app.component.ts = es la logica de renderizado.
          >> import para ser hijo de componente.
          >> @componente para linkear a vista y css
          >> export: para exportar y generar variables/métodos.
      LoQueSeaMierda.app.module.ts = cargar el LoQueSeaMierda.app.component.ts

      * lo importante está en src: 

      el HTML se accede desde el index con la etiqueta: <app-root></app-root>
      Eso hace referencia a la carpeta: app... ahí vamos a ver el archivo: app.component.html
      ahí se renderiza el HTML

Crear nuevas vistas Manual

      crear una carpeta dentro de: app (components)
      Crear la carpeta deseada carpeta adentro de (components)
      //Ejemplo: mi-componente
      Creale estos archivos:
      * mi-componente.app.component.html
        <etiquetas> {{ DATOS }}  </etiquetas>

      * mi-componente.app.component.css
      * mi-componente.app.component.ts

          import { componente } from angular // Osea heredar.

          @component({  selector: 'mi-componente',  template: el.html  })

          export class MiComponente { //todo }



      * registrar el componente en: app.module.ts (root src)
        import { mi-componente } from './components/mi-componente.component'
                                      // carpeta contenedora/carpeta que creamos/ el @component del mi-componente.app.component.ts



        meter ese import { >> mi-componente << } dentro del declaration
                          @NgModule({
                            declarations: [
                              AppComponent,
                              >> mi-componente <<
                            ],
                            ...
                          })

      Listo pongalo en el src\app\app.component.html (app principal)
            basta solo con llamar el selector como etiqueta html 
            <mi-componente></mi-componente>

Crear nuevas vistas Automatico:

CMD (root): ng g component components/otro-componente

Todo el anterior proceso se hace en automatico.

      Para mostrar esos componentes:
      Pongalo en el src\app\app.component.html (app principal)
      basta solo con llamar el selector como etiqueta html 
      <otro-componente></otro-componente>

Instalar Bootstrap

Ir al root del proyecto e instalar el modulo npm de boostrap

CMD: npm install bootstrap jquery @popperjs/core

Ir al archivo: angular.json en el root del proyecto e incluir las librerias. 

      En el root del proyecto hay una carpeta llamada +-node_modules\??? y ahí está el .css y .js 
      Vamos a pegar sus direcciones en el archivo angular.json 

      "styles": [
        "node_modules/bootstrap/dist/css/bootstrap.css",
        "src/styles.css"
      ],
      "scripts": [
        "node_modules/jquery/dist/jquery.js",
        "node_modules/@popperjs/core/dist/cjs/popper.js",
        "node_modules/bootstrap/dist/js/bootstrap.js"
