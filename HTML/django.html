Crear nuevo proyecto: 

      0 -> Crear el entorno virtual:
      Python -m vitualenv env 
      Eso creará una carpeta de nombre env // Con una virtualización del proyecto.

      1 -> Activar el entorno virtual:
      En el CMD ir a la ruta: 
      cd env\Scripts  
      Activar con el comando:
      activate

      2 -> Instalar Django en el entorno virtual:
      Con el CMD ir al root del proyecto e instalar django:
      pip install django
