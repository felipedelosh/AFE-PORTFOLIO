MatPlotLib

Es una libreria de Python para graficar


Instalar: Python -m pip install matplotlib

Usar: import matplotlib.pyplot as plt

IMPORTATE: limpiar el lienzo, los valores se van agregando a la grafica actual.

Ver grafica X vs Y

      x = ["A","B","C","D","E"]
      y = [3,5,4,2,6]

      plt.plot(x, y)
      plt.show()

Ver grafica X,X',X'... vs Y con labels

      x = ["A","B","C","D","E"]
      y2=[5,5,5,7,6]
      y3=[1,1,9,9,0]
      y = [3,5,4,2,6]
      
      plt.plot(x, y, label="y")
      plt.plot(x, y2, label="y1")
      plt.plot(x, y3, label="y3")
      plt.legend()
      plt.show()
