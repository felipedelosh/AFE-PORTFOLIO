NodeJs

Convierte codigo JavaScript a Backend/Desktop DESCARGAR

Se crean archivos.js y se ejecutan como si fueran un programa.

Se creo para tener miles de concurrencias.


Hola mundo

Crea un archivo.js con el siguiente codigo:

      console.log("hola mundo");

Ejecutalo desde el terminal con el siguiente comando:

node archivo.js


Arquitectura de la solución

Los programas se crean en modulos:

        //Crear calculadora
        modulo matematica: math.js 
        modulo main: index.js 
        ----------------------------------------

        //Matematica:
        function add(x1, x2){
          return x1 + x2;
        }
        
        function substract(x1, x2){
          return x1 - x2;
        }
        
        function divide(x1, x2){
          if (x2 === 0){
            return "infinity";
          }
          return x1 / x2;
        }
        
        function multiply(x1, x2){
          return x1 * x2;
        }
        
        exports.add = add;
        exports.substract = substract;
        exports.divide = divide;
        exports.multiply = multiply;

        ----------------------------------------
        //main
        const math = require('./math.js');

        console.log(math.add(2,2));
